Cluster Labels Human Evaluation Framework

Description

Document clustering techniques produce a set of clusters which contain similar documents. However, it requires human intervention to interpret the document clusters. The involvement of human for cluster interpretation makes it impractical when the size of the corpus is huge.

To address this issue, we are working on designing an approach for automatically labeling clusters. Initial experiments were conducted on a dataset of Urdu News Headlines

For the evaluation of the proposed approach, this application requires users to rate the cluster labels produced using 4 different methods. Users are asked to rate the cluster labels according to following aspects:

  1. Completeness: Is the label complete synctactically?
  2. Relevance: Is the label relevant to the content of cluster? For understanding the cluster content, five documents which are nearest to the centroid of the cluster is also presented to the user.
  3. Correctness: Is the label grammatically correct?

For each of the above aspect, the user will rate on a 4-point Likert Scale defined as follows:

Score Description
3 The cluster label is perfect
2 The cluster label is reasonable. A slight improvement is needed.
1 The cluster label is average. An improvement is required.
0 The cluster label is the worst.
